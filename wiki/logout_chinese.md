# [Linux] Bash logout 用法: 退出当前会话

## 概述
`logout` 命令用于退出当前的登录会话。它通常在交互式 shell 中使用，特别是在用户通过终端或控制台登录时。

## 用法
基本语法如下：
```bash
logout [options]
```

## 常用选项
- `--help`：显示帮助信息。
- `--version`：显示版本信息。

## 常见示例
1. **退出当前会话**
   ```bash
   logout
   ```
   这个命令将结束当前的 shell 会话。

2. **在脚本中使用**
   ```bash
   #!/bin/bash
   echo "执行完毕，退出会话"
   logout
   ```
   在脚本的最后使用 `logout` 可以在执行完所有命令后自动退出会话。

## 提示
- 确保在使用 `logout` 之前保存所有未保存的工作，因为该命令将立即结束会话。
- 在使用 SSH 连接时，`logout` 将断开与远程主机的连接。
